# 17-Practice-Questions
Practice Questions

This folder should contain hands-on SQL problems arranged by difficulty.

Suggested categories:

Basic SELECT queries
Filtering with WHERE
Sorting and limiting results
Aggregate functions
GROUP BY and HAVING
All types of JOINs
Nested subqueries
CTE-based problems
Window function challenges
Stored procedure exercises
Trigger-based scenarios
Optimization and indexing questions
Real-world business case studies (e.g., Employee Management, Banking, E-commerce, Hospital Management)

The goal is to strengthen query-writing skills and prepare for interviews and real-world database tasks
